INTRODUCTION
------------

* "Drusys Landing Page builder" module will helps to create responsive layouts.
  It uses the following libraries and licensed under the MIT license.
   - https://github.com/claviska/jquery-minicolors
   - https://github.com/kenwheeler/slick
   - https://github.com/graingert/wow
   - https://animate.style
   - https://getbootstrap.com - Partial bootstrap (Grid & card css only)

REQUIREMENTS
------------

* This module requires paragraphs & media_library (core).

INSTALLATION
------------

* Download the module.
* Copy the drusys_templates & place it to your boostrap 4 theme.
* Install as you would normally install a contributed Drupal module. Visit
 https://www.drupal.org/node/1897420 for further information.

CONFIGURATION
-------------

* Create row & card presets. Create content from Landing Page Builder type.
* http://drupal9.local/admin/structure/ds-lpb/row
* http://drupal9.local/admin/structure/ds-lpb/card
